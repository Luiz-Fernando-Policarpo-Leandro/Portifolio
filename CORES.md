## Teoria das Cores Aplicada

### 1. **Cores Análogas Modernas**
O ciano (#06b6d4) é análogo ao verde no espectro, criando harmonia visual com contraste suficiente para destacar.

### 2. **Vibe Tech/Cyber**
A combinação verde + ciano remete a interfaces futuristas, matrix, e tecnologia.

### 3. **Neutros Frios**
Cinzas frios (#6b7280) mantêm a coerência com a paleta tech.

### 4. **Regra 60-30-10**
- 60% Neutros (preto, cinza)
- 30% Verde (cor primária)
- 10% Ciano (accent/destaque)

- ✅ Backgrounds temáticos (matrix)
- ✅ Gradientes combinados
- ✅ Detalhes de hover
- ✅ Ícones de tecnologias

### Neutros
- ✅ Textos
- ✅ Backgrounds

---

## Dicas de Uso

1. **Manter hierarquia** - ciano para elementos primários, verde para secundários
2. **Usar gradientes** - combinar ciano com verde cria vibe cyber
3. **Respeitar a proporção** - 90% neutros + verde, 10% ciano
4. **Não exagerar no ciano** - usar apenas para destaque

## Como Manter

Ao adicionar novos elementos:
1. Use as variáveis CSS existentes
2. Siga a regra 60-30-10
3. Prefira ciano para elementos de destaque
4. Use verde apenas para efeitos específicos
5. Mantenha a base neutra
